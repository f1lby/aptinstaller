# Kali Linux App Build Script

# A tool to streamline application install for Kali Linux
# This tool uses the APT tool.

### ** Written by F1lby **
Version 1.00 Beta
05 January 2026<br>
<b>As of 10th January 2026, 105 packages are included in the <i>aptpackages.txt</i> file.</b>

Usage Notes:

1) Ensure the file <i>aptinstaller.sh</i> is made executable. This can be done by running the command <i>chmod +x aptinstaller.sh</i>

2) Ensure the supporting file <i>aptpackages.txt</i> is present (this is the reference file for all package names that will be installed. You can amend this to suit your needs.
3) Run the script using the following command line syntax<br>
<i>./aptinstaller.sh</i><br>
<b>or as root</b><br>
<i>sudo ./aptinstaller.sh</i>
